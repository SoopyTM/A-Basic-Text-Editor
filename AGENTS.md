# AGENTS.md

## Build

```bash
make        # builds to output/program
make clean  # removes output/
```

## Run

```bash
./output/program
```

Usage: The program takes one argument - the filename to edit.

## Notes

- Source in `src/`, output goes to `output/`
- Use Google naming conventions: PascalCase for classes, snake_case for functions and variables
